# Yocto-Open Embedded Course


## Structure
* The repo is divided into Exercises and General resources
    * Exercises are divided as per the day of the course
        * Course is currently offered for 3 days.
        * `Day1` and `Day2` require hands on exercises with mild difficulty
        * `Day3` is a task to generate a custom linux os using yocto serving a specific purpose.

## Useful Tip
* `Cheat sheet` can be found in `General Resources`
    * Comprises of 
        * Commands
        * Useful Variables and settings.
        * Tips and tricks

* **Note** : A Virutal Machine with Ubuntu configured with "poky" is required to get started with the course material.
