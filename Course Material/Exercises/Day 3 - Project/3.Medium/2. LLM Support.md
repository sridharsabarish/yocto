# Time for Hallucinations! (Verified!)

* This task aims at creating a personalised LLM capable of running on a RPI!
* We can use `llama.cpp` an open source tool to do this task!


**Tip: Use Chatgpt or Le chat for asking it how to use llama.cpp once you have added the recipe to your linux!**

## Source

 - Github : `https://github.com/ggml-org/llama.cpp`
 - Model : recommended ot use QWEN 3.5 0.8b Q4 GGUF or qwen2.5 0.5b GGUF model, Ask chatgpt or me for the model!

`llama.cpp + model = Local LLM`

### Open Source Model
* Try using a open source model like qwen3.5-0.8b optimized for RPI

* An example way of running your LLM!
