### Shell script

* use chatgpt for inspiration!

## Recipe example

```make

SUMMARY = "Recipe to install my custom file"
LICENSE = "CLOSED"

SRC_URI = "file://" #add your .txt here

S = "${WORKDIR}"

FILES:${PN} += "/opt/myapp" #change accordingly

do_install() {
    install -d ${D}/opt/myapp #change myapp to anything you like
    install -m 0644 ${WORKDIR}/yourtxt.txt ${D}/opt/myapp/yourtxt.txt #change here too

    // Use similar logic to add a shell script to this location


}


```




### 1. Requirements
* In your built linux image, you should be able to go to `/opt/myapp` 
    * Check if the files both the `.txt` and `.sh` are present
    * Invoke this script and see what happens!
            * The script should print a random quote!
