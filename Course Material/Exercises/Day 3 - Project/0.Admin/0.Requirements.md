## Requirements for the project 

1. **Makefile**
    * Targets
        * `build_image`
            * Builds Your Image
                * `bitbake your_image` 
        * `clean_image`
            * Cleans your image
                * `bitbake -c clean your_image`    
        * variable called `HW` that will determine if we build for raspberry or qemu.
            * Makefile should use this variable and set $MACHINE before invoking bitbake! 

        * devtool extract your image! 
2. **Your own Image Recipe**
    * You can take inspiration from `core-image-minimal`  and build your own custom image

3. **Your own Distro Layer**
    * Should be different than Poky

4. **Your own recipe** which installs a file, .c , .txt or .sh into your image under /opt/myproject/
