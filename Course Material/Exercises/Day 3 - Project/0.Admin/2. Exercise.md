# The Project

## Exercises (To get started)

### 1. Building Baseline - RaspberryPI Image
* Change Machine to "raspberrypi4" in local.conf and try to build "core-image-minimal"
* Add support of `wic` in `IMAGE_FSTYPES`, this is needed for raspberry PI!

###  2. FLashing image to Balena Etcher
* Insert micro SD Card into the reader, note you may need to share the flasher.
    * Flash your image to SD card, run as admin if it fails! 
    * Once flashed, Take the SD Card and mount it in RPI and connect microHDMI to your monitor and also connect keyboard! Make sure RPI is powered on!


### 3. Run basic commands
* Make sure that you can access the CLI and run basic commands in the PI!
