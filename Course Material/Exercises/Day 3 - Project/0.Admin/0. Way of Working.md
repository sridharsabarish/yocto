# Project Logistics

# How to approach the tasks
* The taks are classified according to difficulty, start with admin followed by very easy tasks and so on.


## Recommendations
* **Own Image Recipe** : 
    * Recommended to do create your own image and add the newly built recipe as you go further. 
* ** Create a recipe, build it and add to your image**
* Keep verifying your changes in RPI as you keep making progress! and remember to have fun!
* The goal is not to build everything, but to get a feel for RPI HW!
